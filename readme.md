# Polymer Samples

> Polymer samples for current master version of [Polymer](https://github.com/polymer/polymer)

## Run server

Please make sure that NPM has been installed before run gulp

```
gulp serve
```

## Deploy

```
gulp ghpage
```

## Add sample page

```
node ./template.js 'Page title'
```
